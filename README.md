# Olympic Data Analysis Dashboard  
This project is an interactive data analysis dashboard that explores 120 years of Olympic history (1896–2016) using Python, Pandas, Plotly, and Streamlit.
It allows users to visually analyze how the Olympics have evolved — from the number of participating nations and athletes to medal counts and gender participation over time.

## Features
- Medal Tally (Year-wise & Country-wise)
- Overall Olympic Statistics
- Participation Trends (Nations, Events, Athletes)
- Country-wise Medal Analysis & Heatmaps
- Athlete Analysis (Age distribution, BMI trends, Men vs Women participation)
- Height vs Weight scatter plots with medal info

## Tech Stack
- Python, Pandas, NumPy  
- Streamlit  
- Plotly, Seaborn, Matplotlib
